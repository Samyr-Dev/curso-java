# Sobre o projeto

Este repositório tem como objetivo consolidar conhecimentos práticos sobre **desenvolvimento web com Java**, abordando **organização de projetos em camadas**, além do uso de **estruturas sequenciais, funcionais e de streams**.

Também inclui exemplos práticos de **funções e métodos reutilizáveis**, que podem servir de base ou referência para outros projetos.

# Tecnologias utilizadas
## Back end
- Java

## Back end

# clonar repositório
git clone https://github.com/Samyr-Dev/curso-java
